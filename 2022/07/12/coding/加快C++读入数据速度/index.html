加快cin读入数据速度

加快C++读入数据速度

  • 使用 std::ios::sync_with_stdio(false); 语句取消cin 与 stdin 的同步
    • 减少额外的开销
    • 但是禁用后不能再混用cin 和scanf，cout 和 printf
    • 注意要在IO之前解除绑定
  • 使用 tie(0); 解除cin 与 cout 的绑定
