  • int dist[N]; //dist[i]表示到顶点vi的路径长度
    • 如果有边 dist[i] = 边的weight
    • else dist[i] 设置为无穷大
  • T ：保持已经找到了最短路径的顶点的集合
  • 数组dist[]用来保存源点到各个顶点的距离
  • 数组T[]用来保存已经找到了最短路径的顶点的集合
  • 初始化
    • 原点s的weight赋值0
      • dis[s] = 0;
    • 点s能直接到达的边（s,m)
      • dis[m] = w(s,m)
    • s不能直接到达的顶点设置为无穷大
    • 集合T初始时只有顶点s
  • 从dis数组中取最小值就是最短路径
  • 把该点加入到T中就完成了一个顶点
